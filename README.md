# Nitesh Dwivedi Portfolio

A premium recruiter-focused portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Recharts.

## Folder Structure

```text
portfolio3/
|-- index.html
|-- package.json
|-- postcss.config.js
|-- profile2.jpeg
|-- README.md
|-- tailwind.config.js
|-- vite.config.js
`-- src/
    |-- App.jsx
    |-- components.jsx
    |-- data.js
    |-- index.css
    `-- main.jsx
```

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Select the `Vite` framework preset if Vercel does not auto-detect it.
4. Keep the build command as `npm run build`.
5. Keep the output directory as `dist`.
6. Deploy.
